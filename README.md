# E-commerce Analytics Project

## Overview
Analysis of user behavior, funnel conversion, traffic source performance, and campaign effectiveness.

## Dataset
Synthetic multi-table dataset (events, transactions, campaigns, customers, products).

> Raw data is not included due to size limits.  
> Place CSV files into the `/data` folder before running the notebook.

## Key Analyses
- Funnel: view → click → add_to_cart → purchase
- Conversion by traffic source
- Revenue and AOV by traffic source
- Campaign performance

## Key Findings
- Paid Search leads in revenue
- Email shows highest conversion
- Organic/Direct underperform in conversion

- Python (pandas)
- Jupyter Notebook
